### Ajout des icônes dans la barre de navigation

Pour cet ajout, il a fallu modifier le fichier MainTabNav.js se situant dans `app/navigation/`.

Cet ajout est l'import d'icônes provenant [d'ionicons](https://ionic.io/ionicons), sous cette forme :
```js
    const ICONS = {
      OrganisationsTab: 'business',
      ContactsTab: 'person',
      StagesTab: 'school',
    };
```

Ainsi que cette partie du code qui, pour chaque screen énoncé dans la barre de navigation, leur attribue une icône :
```js
    tabBarIcon:({ color, size }) => (
        <Ionicons
        name={ICONS[route.name]}
        size={size}
        color={color}
        />
    )
```

### Ajout du mail de l'utilisateur connecté visible dans l'entête

Pour cet ajout, il a fallu modifier le fichier MaintTabNav.js et OrganisationsStackNav se situant dans `app/navigation/`.

Il a fallu modifier le headerTitle en lui mettant une fonction anonyme qui envoie simplement le titre et le mail de l'utilisateur.

MainTabNav.js :
```js
    headerTitle: () => (
      <View>
        <Text style={{ color: '#fff', fontWeight: 'bold', fontSize: 18 }}>Contacts</Text>
        {user?.email && <Text style={{ color: '#fff', fontSize: 12 }}>{user.email}</Text>}
      </View>
    )

```

OrganisationsStackNav.js
```js
    <Stack.Screen
      name="Organisations"
      component={OrganisationsScreen}
      options={{
        headerTitle: () => (
          <View>
            <Text style={{ color: '#fff', fontWeight: 'bold', fontSize: 18 }}>Organisations</Text>
            {user?.email && <Text style={{ color: '#fff', fontSize: 12 }}>{user.email}</Text>}
          </View>
        ),
      }}
    />
```
