# Introduction

>Le but principal de cette activité est de permettre l'évolution de la base de données et création d’une API REST pour faciliter la saisie des stages par les étudiants et l’utilisation des données par des applications web et mobiles (potentiellement l'AP-41).

## Moyens utilisés

>Afin de faire évoluer la base de données, créer l'API REST et l'interroger, nous utilisons le framework [`PHP Symfony`](https://symfony.com/), qui va nous permettre de créer les tables (appelés Entités) avec leurs attributs respectifs etc.

>Les tests de cet API sont effectués via le framework [`PHP-Unit`](https://phpunit.de/index.html) grâces à ses différents tests unitaires.
